📐 Formulas (Tier 4, all skills Lv10)

Success rate (mithril ore grade=7):
  diff = level × 3 − 7  →  base_val = lookup(diff)
  rate = clamp( (base_val + ⌊TEC/4⌋ + 50) × 0.85, 1, 100 ) + pity_pts/150
  +A and +S: always floor 1% + pity_pts/150

Pity: mithril ore = +4 pts/attempt  ·  150 pts = +1% flat  ·  ~38 attempts/tier

Degrade on fail:  P(degrade|fail) = 0.50 × max(0, 1 − LUK/340)
  LUK=0 (TEC): 50%  |  LUK=255 alone: 12.5%  |  LUK=255 + Anti-Deg item (+75% additive): 0%
Degrade assumptions (user-specified):
Base P(degrade|fail) = 50% for all slot counts. Distribution given degrade occurs:
0-slot: always −1  |  1-slot: 50% (−1) + 50% (−2)  |  2-slot: ~33.3% each of (−1), (−2), (−3)
LUK prevention via LUK/3.4% (Formula A), applied multiplicatively to the 50% base.
This gives 0% degrade at LUK=255 + Anti-Deg, consistent with player experience.
